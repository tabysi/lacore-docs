# Keybinds

These are the **default** keys. Every player can rebind them in
**FiveM → Settings → Key Bindings → FiveM** (the engine remembers your choice).

| Key | Action | Command |
|-----|--------|---------|
| **O** | Open / close the MDT | `/mdt` |
| **I** | Toggle the player list | `/playerlist` |
| **G** | Toggle vehicle engine | `/engine` |
| **K** | Toggle seatbelt | `/seatbelt` |
| **U** | Raise / lower hands | `/hu` |
| **X** | Hand on holster | `/hh` |
| *(unbound)* | Open/close front driver door | `/door 1` |
| *(unbound)* | Open/close front passenger door | `/door 2` |
| *(unbound)* | Open/close trunk | `/trunk` |
| *(unbound)* | Change weapon firing mode | `/firingmode` |
| *(unbound)* | Dismiss a player's report (staff) | `/rd` |

## While hands are up (`/hu` / **U**)
- **Crouch key** → kneel
- **Character-wheel key** → put hands on head

## Holster (`/hh` / **X**)
- Hold **aim** to draw your pistol from the holster
- Tap the character-wheel key to raise/lower the "hold there" hand
- Tap **attack** to cancel

> Anything marked *(unbound)* has no default key — assign one in Key Bindings if you want it on a hotkey, or just use the command.

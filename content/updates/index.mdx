## 3.2.0 — New NUI Phone, Feature Toggles, Air Unit, Corrections & more

Our biggest release yet — a brand-new phone, a fourth MDT skin, real framework
support, four new gameplay systems and a full security overhaul.

- **📱 All-new NUI Phone** — built from scratch: calls, SMS, **911/311**, camera,
  Bleeter feed, messenger groups, ringtones and 7+ apps.
- **🖥️ CAD / MDT** — a new **9100-T Retro Terminal** (a 1990s green-CRT MDT with a
  clickable on-screen keyboard, now four terminal skins), the person record
  redesigned as an **RMS-style folder** with registered vehicles, a mobile
  **fingerprint scanner**, a darker CAD theme and an unread-message badge.
- **🔗 Real framework integration** — an **ESX / QBCore / QBox** bridge: name, job,
  characters, vehicles and **licences** sync automatically, and framework jobs go
  on duty without running a second identity system.
- **🚔 New gameplay systems** — **Corrections / Jail**, **Impound / Tow**,
  **Air Unit** (heli-cam) and a full **K9** overhaul.
- **🛡️ Security & core** — a server-authoritative **anti-cheat overhaul** with
  near-zero false positives, **global-ban network** protection, per-module
  **feature toggles**, an own **NativeLacoreUI** (NativeUI dependency dropped),
  config backup/restore and **EN / DE / RU** localisation.

Plus dozens of fixes — framework garage-plate compatibility, a vehicle-spawn crash
fix, CCTV improvements and more. See the full page for the complete list.

## 3.1.5 — CCTV Surveillance Suite, Scanner & Field Cameras

The CCTV system became a full surveillance suite.

- **CCTV camera viewer** (`/cctv`) — fixed-position POV camera with real area streaming, gated by
  network + job.
- **Targeting scanner** — click-to-scan people and vehicles with an animated scan sequence, a
  compact **mugshot** and an **MDT summary** (priors / BOLO / recent records).
- **Field cameras** — `/camtool` places cameras at runtime, `/cameras` manages them; they persist
  and show up in `/cctv` and on the dispatch map.
- **Dispatch-map cameras** — 📹 markers with a "view this camera?" modal that opens the live view
  in front of the console and reopens the console on exit.
- **Turf is now opt-in** (`Turf.enabled = false` by default).
- Fixes: no firing while operating a camera, dispatch console foreground handling, tighter map
  position tracking.

# Full Changelog

Notable changes per release. The complete, line-by-line changelog ships with the resource in
`changelog.md`.

## 3.1.5 — CCTV Surveillance Suite, Scanner & Field Cameras

### Added — CCTV camera viewer (`/cctv`)
- Fixed-position POV camera (WASD / arrows look, scroll zoom) with real area streaming (clone ped +
  teleport). Cameras grouped into networks, gated by job (`leo` / `security` / `all`).
- You cannot fire your weapon while operating a camera.

### Added — Targeting scanner
- Left-click raycasts a **person** or **vehicle** and runs an animated scan sequence (reticle,
  sweeping line, live percentage). Duration `CCTV.scanTime` (default 6 s).
- An info card follows the target on screen with a compact **mugshot** and, for registered people,
  an **MDT summary** (priors, active BOLOs, recent records). Flags: ACTIVE WARRANT, BOLO ACTIVE,
  REPORTED STOLEN, UNREGISTERED. NPCs / unregistered targets resolve locally so a scan never hangs.
- The operator's desk standin is a networked ped that resolves to the real player when scanned.

### Added — Field cameras + dispatch-map integration
- `/camtool` places cameras by shooting a spot; `/cameras` manages them (rename / delete / view /
  toggle tool). Placed cameras persist (`data/cctv_cams.json`) and appear under a "Field Cameras"
  network.
- The dispatch map shows all accessible cameras as 📹 markers; a "view this camera?" modal opens
  the live view and reopens the console on exit.

### Changed
- **Turf / Gang-War is opt-in** — `Turf.enabled = false` by default; `Turf.drawMarkers` off too.

### Fixed
- Could shoot while inside a camera; dispatch console stayed in front of the live view; dispatch
  map position tracking tightened to ~1 s updates.

## 3.1.3 — Rebranding to LACORE + release-ready

- **Breaking:** resource rename `pvp-corev3` → `lacore`; convars `pvp_*` → `lacore_*`; ACE groups
  `group.pvp_*` → `group.lacore_*`; command/keybind renames. Automatic DB migration
  (`pvp_core_store` → `lacore_core_store`, `pvp_logs` → `lacore_logs`).
- Connect-hang fix ("Checking bans..."), post-launch security hardening (turf exploit, admin
  spectate / noclip cleanup), IP-lock enforcement, complete `DOCS.md` + `LICENSE.md`.

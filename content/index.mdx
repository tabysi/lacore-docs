# LACORE

**LACORE** is an all-in-one, configurable **roleplay core for FiveM**, created by
[@tabysi](https://github.com/tabysi). It bundles a complete, production-ready law-enforcement suite
(MDT/CAD, dispatch console, CCTV surveillance), a civilian roleplay layer, persistent multi-character
profiles, an admin toolkit, and a hardened security stack — all in **one cohesive resource**.

LACORE is a **configurable core, not a bare framework**: every feature works out of the box and is
tuned through config files and convars. It runs standalone with its own identity layer, or bridges
to **ESX / QBCore / QBox** when you already run a framework.

<Callout type="info">
  All in-game text is English and fully localised (`en` / `de` / `ru`). This documentation covers

## Feature overview

### Law enforcement
- **Mobile Data Terminals** — four skins on one shared backend: **LAPD** (PremierOne), **LASD**
  (CAD/PCMS), **Agency** (CHP-styled re-skin for every other department), and **EMS/Fire**.
- **Dispatch console** (`/dispatch`) — a live tile map of incidents, units and callers, with zone
  polygons, districts, BOLOs, charges, a dispatcher ⇄ unit workflow, and CCTV cameras on the map.
- **CCTV surveillance** (`/cctv`) — a fixed-position camera viewer with a **targeting scanner**
  (mugshot + MDT summary), **runtime-placeable field cameras**, and dispatch-map integration.
- **Evidence & reports**, **BOLOs**, **charges & citations** driven by a configurable penal code.

### Civilian & identity
- **Multi-character profiles** — persistent characters ("digital file"), records, vehicles and
  achievements.
- **Civilian layer** — emote radial (bridged to DPEmotes / RPEmotes), props, digital ID,
  self-service vehicle registration, and an org/gang framework with optional turf war.

### Operations & security
- **Admin menu** (`/admin`) — server-authoritative moderation + world controls.
- **Big Brother** (`/bblog`) — admin action logging with Discord webhooks.
- **Security** — IP-lock license enforcement, event-honeypot anticheat, permissions (ACE + Discord
  role bridge).
- **Web dispatcher** — optional external browser dispatcher via a FiveM ↔ Node bridge with Discord
  OAuth.

## Requirements

| Requirement | Notes |
| --- | --- |
| FiveM server | `fx_version cerulean`, `game gta5`. |
| [oxmysql](https://github.com/overextended/oxmysql) | Persistence. Falls back to local JSON if unavailable. |
| MySQL / MariaDB | LACORE creates its own tables automatically. |
| DPEmotes / RPEmotes *(optional)* | For the civilian emote bridge. |

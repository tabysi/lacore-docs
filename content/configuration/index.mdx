# Configuration

LACORE is configured in three layers:

1. **Convars** in `server.cfg` — secrets (Discord token, webhooks) and a few runtime toggles.
2. **`config.lua`** — the main gameplay/world config (agencies, call center, time, transit, …).
3. **`configs/*-sh.lua` / `*-sv.lua`** — per-feature config files.

<Callout type="info">
  All files in `configs/` and `config.lua` are **escrow-ignored** — you can open and edit them
  freely. Convars keep secrets out of the code and out of client downloads.
</Callout>

## Localisation

All user-facing strings are localised:

- **Lua** (server/client): `T("key")` → `lang/en.lua` (+ `de.lua`, `ru.lua`).
- **NUI**: `t("key")` → `lang/en.json` (+ `de.json`, `ru.json`).

The active language is set by `Language` in `config.lua`; the MDT/NUI can use a separate language via
`MdtLanguage`. To add a language, copy the `en` files, translate, and register the code in
`configs/locale-sh.lua`.

# LACORE Products

Standalone LACORE resources you can run **alongside LACORE Core** (or on their own).
Each is a focused, self-contained script — drop it in, configure it, done.

<Callout type="info">
  Looking for the **standalone MDX / CAD**? That's [LACORE MDT](/lacore-mdt). This page collects the
  *gameplay* add-ons.
</Callout>

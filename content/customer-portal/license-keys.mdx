# License keys

A **license key** links a running server to your dashboard account. Once linked, the server shows up
under [Servers](/customer-portal/servers) and can pull its [config](/customer-portal/server-config)
online.

<Steps>

### Create a key

On **Dashboard → License keys**, click **Create key**. Give it a label (e.g. "Main RP server") so you
can tell your keys apart.

### Paste it into server.cfg

Copy the ready-made line and drop it into your server's `server.cfg`:

```cfg
setr lacore_license_key "LACORE-XXXXX-XXXXX-XXXXX"
```

### Start the server

Within a minute of starting, the server appears under **Servers**, verified as yours.

</Steps>

<Callout type="info">
  **Revoking a key.** Revoke turns a key off without deleting it — any server still using it simply
  becomes "unlicensed" on its next heartbeat (it keeps running on its local config). Create a fresh key
  to re-link.
</Callout>

<Callout type="warning">
  A key is what proves a server is *yours*. Treat it like a password — don't share it publicly.
</Callout>

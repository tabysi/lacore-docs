# Store screenshots

Drop Tebex / Cfx.re store images here, then run `npm run deploy` from `website/`.

Each file becomes publicly reachable at:

    https://tabysi.github.io/lacore-docs/store/<filename>

Use that URL in the `[img]` (BBCode) or `<img src>` (HTML) tags of the Tebex
product description.

Suggested files (referenced by the description templates):

- `banner.png`   — wide hero banner (1920×~600)
- `overview.png` — feature overview / collage
- `mdt.png`      — MDT / CAD screenshot
- `dispatch.png` — dispatch console
- `civilian.png` — civilian layer / profile

Keep images reasonably sized (Tebex renders them at content width). PNG or WEBP.
